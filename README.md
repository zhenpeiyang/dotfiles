# Dotfile configurations

Features:
- `setup.sh` links dotfiles, backs up existing files, installs vim-plug, oh-my-zsh, and tmux TPM, then runs `PlugInstall`.
- `zsh` setup with Homebrew-first PATH, sane history settings, vcs_info prompt (lane/repo/branch), nvm loading, and convenience aliases.
- `tmux` config with vi-style keys, Alt+arrow pane navigation, styled status bar, and TPM plugins (`tmux-yank`, `tmux-resurrect`).
- `vim` config with gruvbox theme, indentation and search defaults, clipboard integration, tab mappings, and vim-plug plugins (CtrlP, markdown preview).
- Included colorschemes live under `.vim/colors`.
